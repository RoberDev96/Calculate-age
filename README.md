# 🎂 Calculadora de Edad

Una aplicación web simple y efectiva que calcula la edad exacta de una persona en años, meses y días a partir de su fecha de nacimiento.

## ✨ Características

- Cálculo preciso en años, meses y días.
- Interfaz limpia, responsive y fácil de usar.
- Validación de fechas (no permite fechas futuras).
- Tecnologías web estándar: HTML5, CSS3 y JavaScript puro.
- **Desplegada y funcionando** en GitHub Pages.
